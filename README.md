# Personal Website - Jaslyn Pan

A modern, responsive personal portfolio website built with React and Vite.

## Features

- **Responsive Design**: Optimized for all devices
- **Smooth Animations**: Powered by Framer Motion
- **Modern UI**: Clean, professional aesthetic with Tailwind CSS
- **Bilingual**: English and Chinese language support

## Tech Stack

- **Framework**: React 18 + Vite
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Deployment**: Vercel

## Live Site

Visit: [https://personalweb-sigma-woad.vercel.app](https://personalweb-sigma-woad.vercel.app)

## Local Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Project Structure

```
src/
├── components/     # React components
├── contexts/       # Context providers (Language, etc.)
├── data/          # Resume data and content
└── App.jsx        # Main app component
```
